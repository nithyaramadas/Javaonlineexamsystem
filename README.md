Online Exam System with Question Bank Generator (Java)

This project is a Java-based Online Exam Management System that allows administrators to create exams, generate randomized question papers, and manage user assessments efficiently. Students can take exams through a graphical user interface, and results are automatically calculated and displayed.

The system includes a Question Bank Generator feature that stores and retrieves questions dynamically, reducing manual effort and ensuring balanced question distribution.

 Features:

User Roles

Admin: Manage question bank, create exams, view results.

Student: Attempt exams, view scores and feedback.

Question Bank Generator

Add, update, and delete questions.

Supports multiple-choice questions with correct answers stored securely.

Auto randomization of questions during each exam.

Exam Module

Timer-based examination interface.

Automatic evaluation and instant result display.

Shuffled questions to avoid cheating.

Result System

Stores student attempts and results.

Generates analytics: score, percentage, and pass/fail status.

Secure Data Handling

Basic authentication for login.

Local file or database storage (as implemented).

User-Friendly Interface

Built with Java Swing for an interactive desktop UI.

 Technologies & Tools Used
 
Programming Language :	Java
GUI Framework	: Java Swing
IDE Used :	Visual Studio Code

 How to Run

Install Java JDK 8+

Clone the repository:

git clone <repository-link>


Open project in VS Code

Compile and run:

javac Main.java
java Main
